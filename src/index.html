About Me:

I'm a full-stack developer skilled in JavaScript/TypeScript, NodeJS, React, HTML, CSS, MongoDB, PostgreSQL, NextJS, Linux, Nginx, Git and beyond. I am passionate about developing clean and intuitive user interfaces that offer an engaging user experience.

This portfolio site was not made with a front-end framework due to the need for hosting demo projects locally that were created with vanilla JavaScript. The projects have been changed slightly from production for demonstrative purposes.
